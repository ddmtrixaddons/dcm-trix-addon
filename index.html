DCM Trix Addon - Google Sheets Addon

Overview

The DCM Trix addon is an open source addon for Google Sheets that allows users to download and sync reports from DoubleClick Campaign Manager (a.k.a. Campaign Manager) on a schedule. The original source code is available at https://github.com/google/dcm-trix-addon. The published addon uses this code as is without any changes.

Questions/Problems?

For any issues, please raise a bug tracker on the original github project or raise a post on the public forum. Note that support provided is on a best effort basis.
